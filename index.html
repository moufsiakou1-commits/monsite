💡
DataSciLab accepte CSV et Excel. Convertissez votre fichier en quelques secondes avec les instructions ci-dessous.
Convertir depuis SPSS
Méthode 1 — Menu SPSS :
1. Ouvrez votre fichier .sav dans SPSS
2. Cliquez sur Fichier → Exporter → CSV
3. Choisissez l'emplacement et cliquez OK
4. Importez le fichier CSV dans DataSciLab ✅
// Méthode 2 — Syntaxe SPSS
SAVE TRANSLATE OUTFILE='monfichier.csv'
  /TYPE=CSV
  /FIELDNAMES
  /CELLS=VALUES.
EXECUTE.
Convertir depuis Stata
Méthode 1 — Menu Stata :
1. Ouvrez votre fichier .dta dans Stata
2. Cliquez sur File → Save As → CSV
3. Sauvegardez et importez dans DataSciLab ✅
// Méthode 2 — Commande Stata
use "monfichier.dta", clear
export delimited "monfichier.csv", replace
Convertir depuis R
Copiez-collez ces quelques lignes dans votre console R :
// Depuis un fichier .rds
data <- readRDS("monfichier.rds")
write.csv(data, "monfichier.csv", row.names=FALSE)
// Depuis un objet R existant
write.csv(mon_dataframe, "monfichier.csv", row.names=FALSE)
# Ou avec le package haven pour lire .sav/.dta :
library(haven)
data <- read_sav("monfichier.sav")
write.csv(data, "monfichier.csv", row.names=FALSE)

ℹ️ À propos de DataSciLab

🧬
Notre Mission

DataSciLab rend l'analyse statistique et scientifique accessible à tous — étudiants, chercheurs et professionnels de santé en Afrique et partout dans le monde. Gratuitement, sans compromis sur la qualité.

🌍
Notre Vision

Devenir la plateforme de référence en analyse de données scientifiques pour les pays francophones d'Afrique, en éliminant les barrières financières et techniques qui freinent la recherche.

⚖️ Modèle économique transparent
DataSciLab est 100% gratuit pour tous les utilisateurs. La plateforme est financée par :
🎓 Partenariats avec des universités
🌍 Subventions et ONG de recherche
🏛️ Versions institutionnelles (les universités paient, les étudiants accèdent gratuitement)
✅ Aucune publicité intrusive. Vos données ne sont jamais vendues.
💡 Des limites techniques raisonnables s'appliquent pour garantir la stabilité du service.

🔒 Politique de confidentialité

1. Stockage des données

Toutes vos données (compte utilisateur, datasets, analyses, notes) sont stockées uniquement dans votre navigateur via localStorage. Aucune donnée n'est transmise à nos serveurs.

2. Clé API Anthropic

Votre clé API Anthropic est stockée localement dans votre navigateur. Elle est utilisée uniquement pour les appels directs à l'API Anthropic depuis votre appareil. Nous n'y avons pas accès et ne la stockons jamais sur nos serveurs.

3. Fichiers importés

Les fichiers que vous importez (CSV, Excel) sont traités entièrement dans votre navigateur. Ils ne sont jamais envoyés à des serveurs tiers. Lorsque vous fermez ou actualisez la page, les données importées sont effacées de la mémoire.

4. Pas de publicité, pas de tracking

DataSciLab ne contient aucune publicité et n'utilise aucun outil de tracking (Google Analytics, pixels publicitaires, etc.). Votre activité sur la plateforme reste strictement privée.

5. Vos droits

Vous pouvez supprimer toutes vos données à tout moment depuis les Paramètres → "Effacer toutes les données". Pour toute question relative à la confidentialité : moufsiakou1@gmail.com
